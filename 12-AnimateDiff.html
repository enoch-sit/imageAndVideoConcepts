01

The Motion Module Concept

AnimateDiff's core idea: freeze the entire SD U-Net, then insert "motion modules" — temporal self-attention layers — at the same resolution levels as the spatial attention. The motion modules are the only parameters trained.

Interactive: U-Net Block with Motion Module
Click a layer to see what it processes. The motion module inflates the 2D attention to attend across the time dimension.

Single-Image → Multi-Frame Inflation

For image generation, the U-Net processes tensors of shape (B, C, H, W). For video, the batch dimension is repurposed as the frame dimension. Each "video batch" has shape (B·F, C, H, W) where F is the number of frames (16). Spatial attention operates on (B·F, HW, C) — each frame independently — while the motion module reshapes to (B·HW, F, C) to let attention flow across frames at each spatial position.

Tensor reshaping for temporal attention Spatial: (B·F, H·W, C) → attend across space
Temporal: (B·H·W, F, C) → attend across frames
F = 16 frames (AnimateDiff default)
02

Temporal Self-Attention

The key component of each motion module is temporal self-attention: at each spatial position (h,w), the F feature vectors (one per frame) attend to each other using standard multi-head attention with rotary positional embeddings along the time axis.

Drag the frame slider to see which other frames it attends to. Brighter = stronger attention weight (illustrative).
7

Rotary Position Embedding (RoPE) along Time

Unlike spatial 2D position embeddings, the temporal axis uses rotary positional embeddings (RoPE-1D) applied to the Q and K vectors. This allows the motion module to generalise to different frame counts at inference — it was trained on 16 frames but can generate longer clips by extending the time axis.

AnimateDiff-Lightning variant
AnimateDiff-Lightning applies the same adversarial distillation from SANA-Sprint to the temporal motion modules: using a video discriminator, the 16-step video denoising is compressed to 1–2 steps. This is what enables near-real-time video preview generation in this course's pipeline.

Composability

  • + ControlNet: per-frame skeleton/edge ControlNet ensures the video follows a specific pose sequence (used for human animation in this course)
